# Linux Section 1: Linux 系统基础和简单命令行操作

## 小节信息

- **课时**: 1～2 学时
- **难度**: 入门级
- **前置要求**: 基本计算机操作能力, 完成 [虚拟机安装](../README.md)

## 学习目标

完成本节课后，学员将能够：

1. 理解 Linux 操作系统的基本概念和特点
2. 独立安装和配置 Ubuntu 系统
3. 熟练使用 Linux 桌面环境
4. 掌握命令行基本操作
5. 进行文件和目录管理
6. 使用包管理器安装软件
